A Small Story...

We'll begin with a story. Imagine that you are living in the age of the Early Man! There are no numbers. All that you know of counting are the following symbols :

  • @ - which means
  • # - which means 'one'
  • $ - which means 'two'

One day, you collect some apples scattered down under an apple tree. If you like to express the quantity of apples using the notation available (listed above), how would you do that? If there were 8 apples, how would you go about using the notation? if I were you, I would have grouped the apples into groups of three, say like this :

                             

Here, two groups are total (3 in a group) and one is partial with 2 apples. Now, to represent the number of apples, I'd first write the number of total groups, followed by the number of apples in the partial group, like this - $$. Therefore, '$$' successfully tells that there are 8 apples.

If there were 14 apples, how would you express the number?

                               

Here, the no. of total groups = 4 and the no. of apples in the partial group = 2. The number '4' can be represented as '##' (this is nothing but grouping the groups again into 1 total and a partial group with 1 apple).

                                                                           

Here, the no. of total groups = 4 and the no. of apples in the partial group = 2. The number '4' can be represented as '##' (this is nothing but grouping the groups again into 1 total and a partial group with 1 apple). .

Hence, 14 can be represented as '##$' (4 groups + 2 in the partial group)

If there were 27 apples, how would the representation look like? Take some time and try out. To what level of grouping of the groups does it lead to? Did you find any difficulty in the grouping. What is the role of zero in such a representation? Think about it.

Actually, the answer is ' #@@@'. Did you observe that in the process of grouping, not once did we get a partial group? How should such absence of partial groups be represented? Obviously, we need some notation to denote absence. That is zero. Do you now appreciate the discovery of zero by the Indian Mathematicians!



Place Value System of Numbers

The above exercise was just to introduce the place value system of numbers. In our day-to-day representation of numbers, we follow the place value system (PVS). In PVS, base is defined as the no. of symbols used for the representation. For instance, the base is 3 in the above example. We represent numbers greater than or equal to the base as a sum of consecutive powers of the base.

Let us see for example, the PVS used in decimal numbers. Here there are ten symbols (0-9). Hence the base is 10. Consider the number 548. It is actually 8 x 1 + 4 x 10 + 5 x 100. Are you able to relate this to the units, tens and hundreds place which you studied in your lower classes? Instead of 10 symbols, if we had 3 symbols only, we go back to the above example. Instead of '@' ,'#' and '$', let us use the digits 0-2. Then, the representation of fourteen would be 112 ( 2 x 1 + 1 x 3 + 1 x 9 ). Try it out similarly for eighty one.

Formally, PVS representation of a number is

( an an-1 an-2 ... a1 a0 ) = a0b0 + a1b1 + a2b2 + ... + an-1bn-1 + anbn

where 'ai ' is a digit ( which can vary from 0 to b-1) in the representation and 'b' is the base. This definition of PVS can also be extended with decimal points.

( an an-1... a1 a0 .a-1 a-2 ... a-m+1 a-m ) = anbn + an-1bn-1 + ... + a0b0 + a-1b-1 + ... + a-m+1b-m+1 + a-mb-m

Now, you may wonder why all this is being dealt with in a course of Indian Mathematics. The answer is that, all the shortcuts and various arithmetic methods of Indian Mathematics are based on the PVS of numbers.

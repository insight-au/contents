Solutions to exercises in trigonometry :
  • Given that the tribhujank for 30 degrees is sqrt(3), 1, 2, find the Tribhujank for 45 0 , 600,   90 0 and 180 0 .

    30 sqrt(3) 1 2
    30 sqrt(3) 1 2
    60 2
    1    		 2 sqrt(3)
    sqrt(3)    		 4
    2

The tribhujank of 90 0  can be found by adding 30 & 60. The value is 0, 1, 1.
The tribhujank of 45 0 can be found by finding the tribhujank of half of 90. The value is 1,1,sqrt(2).
The tribhujank of 180 0 is found by finding the tribhujank of double 90. The value is -1, 0, 1.

  • Prove that (sin A + sin 2A) / (cos A - cos 2A) = cot (A/2)
    Let tribhujank of A be x, y, 1.
    Then, the tribhujank of 2A is (x2 - y2), 2xy, 1.
    The tribhujank of (A/2) is (x+1), y, sqrt((x+1)2 + y2)

    sin A = y ; sin 2A = 2xy
    cos A = x ; cos 2A = (x2 - y2)
    cot (A/2) = (x + 1) / y.

    LHS is :
    (y + 2xy) / ( x - (x2 - y2))
    = y (1 + 2x) / (x + 1 - 2x2 )     as  y2 = 1 - x2
    = y (1 + 2x) / (1 + 2x) (1 - x)
    = y / (1 - x)
    Now, multiply & divide by (1+x)
    LHS = y (1+x) / (1-x)(1+x)
    = y (1 + x) / ( 1 - x2) = y (1+x) / y2
    = (1 + x) / y , which is cot (A/2) = RHS.
